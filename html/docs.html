WhatsApp Text Formatting

You can format your WhatsApp messages using special characters. Here are some examples:

Bold

To make text bold, wrap it with asterisks:

*bold text*

Result: bold text

Italic

To italicize text, wrap it with underscores:

_italic text_

Result: italic text

Strikethrough

To strikethrough text, wrap it with tildes:

~strikethrough~

Result: strikethrough

Monospace

For monospace text, use three backticks:

```monospace```

Result: monospace

Lists

You can create bullet point lists using dashes:

- Item 1
- Item 2
- Item 3

Result:

  • Item 1
  • Item 2
  • Item 3

Combining Formats

You can combine different formats:

*_Bold and italic_*

Result: Bold and italic

Line Breaks

To add a line break, use a backslash followed by 'n':

Line 1\nLine 2

Result:

Line 1
Line 2

Important Note

Make sure to use plain text when sending formatted messages. Some special characters might not work as expected if you use rich text or HTML formatting.
